Coding Projects Website

A fun look into some past coding projects, including interactive elements!

Use the up and down arrow keys to move through the website!

Python Website: Study Spot

Using Django, a group of 4 fellow students and I created a college website group study spot site. It allows users to create study meetups with many attributes such as amount of people, location, time, and type of study. It also allows users to join those groups, view study locations on a map, and create profiles.

Python Website: Georgia Tech Movie Store

Using Django, a group of 4 fellow students and I created a movie store. It functions similarly to a real store, with reviews, the ability to purchase items, and create accounts. There is also admin functionality to create new movie listings and adjust prices.

Python Website: Breath of the Wild Compendium

Using public API and Streamlit in CS 1301, I was able to make a useful website that displays weapons, items, etc. and tell stats about them in The Legend of Zelda: Breath of the Wild. I included 3 search features: by name, by item number, or randomly by category.

HTML Website: Harry Potter/Georgia Tech Sorting Quiz

I created a "Buzz Feed" style form in CS 1301 that would ask questions about Georgia Tech related things and sort the user into a Hogwarts house. I used different html form features and if statements to determine the house.

AI Chatbot 1: BuzzBot

In a small group in LMC 2700, we created an AI Buzz themed chatbot with help from AI. The goal of BuzzBot is to help new students with questions about Georgia Tech. However, this iteration fell short with accuracy due to the lack of direct references. This was fixed in the Judicial AI project.

AI Chatbot 2: Judicial AI

In a three person group with the help of AI in LMC 2700, we created a Judicial AI chatbot. The purpose is to speculate on a possible future of the legal system in the United States. The current iteration takes in files of United States general law, and gives advice based upon that information. It uses OpenAI to give accurate responses and communicate with the user.

Video Game: Penguin Mechanic

For Intro to Game Studies, I was tasked to make a game based around a mechanic. I included coin collecting, a counter, and movement.

Video Game: Galactic Robotz

Using an online course in high school, I was able to design and partially program a platformer game in Unity. It utilizes enemies, collectables, and other platforming elements. It has some bug problems due to the hardware limitations I had at the time.

Program: Atlanta Braves Lightsaber

The coding was light for this, only having to add in a prebuilt line of code to determine the color and timing. I converted videos and sound bites into .wav files to the specific kbps needed by the lightsaber using DiVinci Resolve.
